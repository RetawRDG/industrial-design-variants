# Limpeks SEO next steps

## Current safe baseline

- Static pages have title, description, canonical, Open Graph, Twitter card and JSON-LD for Organization/WebSite/WebPage/BreadcrumbList.
- `robots.txt` exposes the sitemap.
- `sitemap.xml` lists the current public static pages.
- No fake Product/Offer schema is added because current pages are catalog/listing pages and public price/availability are unknown.

## Next real SEO work

1. Create one crawlable HTML page per important product/category, for example `/parts/zatvory-dn300/`, `/parts/vibratory-omb/`, `/parts/filtry-bsu/`.
2. Give each page visible specs table, real photos, request button, and exact unknowns marked as unknown.
3. Add `Product` schema only on single-product pages. Add `Offer` only when price or availability is real and current.
4. Add diagnostic articles: `не открывается затвор БСУ`, `не работает вибратор бункера`, `пылит фильтр цемента`, with links to relevant parts.
5. After DNS approval, replace canonical/sitemap base URL with `https://sale.limpeks.ru/` and add Search Console/Yandex Webmaster.
